# mk-cn-resource

## 项目简介

本项目是一个面向中文互联网资源的 HTML 页面归档仓库。我们收集并整理多个独立、自包含的 HTML 文件，用于保存特定主题或功能的网页内容。这些页面不针对任何单一域名或具体网站，而是作为资源集合，供后续查阅、参考或离线使用。

## 目录说明

```
mk-cn-resource/
├── pages/           # 存放归档的 HTML 页面文件
├── assets/          # 页面所需的资源文件（如图片、样式、脚本等）
├── index.html       # 项目首页（可选）
└── README.md        # 本文件
```

- **pages/**：每个独立 HTML 页面放置在此目录下，文件名应反映内容主题。
- **assets/**：若页面依赖外部资源（如 CSS、JS、图片），请统一存放于此，并在页面内使用相对路径引用。

## 页面归档说明

- 本仓库中的 HTML 页面均为静态、自包含或轻度依赖本地资源的文件。
- 每个页面均保留其原始结构与功能，但可能移除动态外部请求或追踪代码。
- 页面内容以中文为主，部分可能包含其他语言资料。
- 归档页面不代表对其中内容的认可或推广，仅作为资料保存。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来补充、更新或修正页面内容。
- 新增页面请放置于 `pages/` 目录，并确保文件命名清晰、无冲突。
- 若页面依赖外部资源，请尽量将其本地化至 `assets/` 目录。
- 本仓库不承诺持续更新或长期可用性，但会尽力保持内容稳定。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件。
3. 或通过本地 HTTP 服务器运行（如 Python `http.server`）以支持某些页面的跨域资源加载。
